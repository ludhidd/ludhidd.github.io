git语法简记

# 基础处理

    1.设置git信息：
        git config --global user.name " *** "
        git config --global user.email ***
    2.git帮助手册
        git help config
    3.git初始化
        git init
    4.克隆git仓库
        git clone ***
    5.检查git配置信息
        git config --list

# 文件处理

    1.添加文件：把文件添加到缓冲区（和后面加“.”的区别在于，加all可以添加被手动删除的文件，而加“.”不行）：
        *  git add filename
        *  git add .
           git add --all
    2.删除文件
        git rm filename

# 查看状态

    1. 查看当前文件状态
        git status -s
    2.查看尚未暂存的文件更新
        git diff
    3.查看已暂存的将要添加到下次提交里的内容
        git diff --staged
    4.将暂存区的内容提交到当前分支
        git commit -m "..."
    5.查看提交历史'(-p，显示用来每次提交的内容差异,-2显示2次提交)'
        git log (-p) (-2)
    6.版本回退
        1.回退到上一个版本'（HEAD代表当前版本，有一个^代表上一个版本，以此类推）'
            git reset --hard HEAD^
        2.回退到指定版本'(其中d7b5是想回退的指定版本号的前几位)'
            git reset --hard d7b5

# git远程仓库的使用

    1.查看远程仓库'(-v,显示需要读写远程仓库使用的 Git 保存的简写与其对应的 URL)'
        git remote -v
    1.添加远程仓库
        git remote add pb ***
        git remote -v
    2.推送到远程仓库'(推送其他分支，比如dev，就改成：git push origin dev)'
        git push origin master

# 分支处理

    1.创建分支
        git branch ***
    2.分支切换
        git checkout ***
    3.创建分支并切换到创建的分支
        git checkout  -b ***
    4.合并某分支的内容到当前分支
        git merge ***
    5.拉取分支
        git pull
    6.删除分支
        git branch -D ***
    7.删除远程分支
        git push origin -D ***
